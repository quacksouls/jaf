How to run code examples

The code examples in this book are usually presented as complete scripts. You should be able to run an example script as is with minimal changes. Each example script can usually be run in a web browser.

Your web browser

You should be able to run each code example in your web browser. A simple way to run a script in your web browser is to embed the script inside an HTML document. Consider the following template for an HTML document:

template.html
<!DOCTYPE html> <html> <head> <title>Demo</title> </head> <body> <script> // Insert JavaScript code here </script> </body> </html>

Suppose you want to run the following script in your browser:

me.js
const me = "I. M. Learner"; console.log(me);

Insert the JavaScript code at the relevant place in the HTML template. Save the modified template as an HTML file, e.g. me.html, to result in the following file:

me.html
<!DOCTYPE html> <html> <head> <title>Demo</title> </head> <body> <script> const me = "I. M. Learner"; console.log(me); </script> </body> </html>

Open the file me.html in your web browser, then load the developer console of your browser. On Firefox, pressing the key F12 loads the developer console. Refer to the documentation of your web browser for how to load the developer console corresponding to your browser. Online IDE

An online Integrated Development Environment (IDE) is a browser-based IDE that supports the following features (and perhaps more):

  • Text editor
  • Syntax highlighting for various programming languages
  • Run your source code in the browser
  • Some form of version control
